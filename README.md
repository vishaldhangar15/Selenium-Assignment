
# Selenium-Assignment

Flipkart Automatoin using Selenium with the help of Node.JS

**To run the Project**
- Clone the repo from [Here](https://github.com/vishaldhangar15/Selenium-Assignment)!
- run 
    npm install 
- run 
    node test.js
- the output list is displayed in terminal
